# NBA Analytics Pipeline

Repositorio colaborativo para el análisis de datos de la NBA:
- Ingesta y limpieza de datos
- Análisis exploratorio
- Modelos de predicción
- Dashboards y visualizaciones

## Estructura

- `data/raw/`: datos crudos (no se suben al repo)
- `data/processed/`: datos limpios (no se suben al repo)
- `notebooks/`: notebooks para exploración
- `src/`: código fuente (paquete Python)
- `tests/`: pruebas
